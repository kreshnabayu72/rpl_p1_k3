NYALAIN REACT
1. Buat folder (misal x)
2. buka folder x di vscode
3. buka terminal vscode & ketik:
	a. npm install
	b. npm install create-react-app
	c. npx create-react-app nama-app (agak lama)
	d. npm install react-router-dom 
	e. cd nama-app
	f. npm start
	
4. ganti folder src & public pake folder src & public yang ada di github
